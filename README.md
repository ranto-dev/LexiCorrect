# LexiCorrect

## Documentation

Pour l'utiliser en local, suivez les etapes suivante:

- Commencer par le telechargement du [projet](https://github.com/ranto-dev/LexiCorrect.git) ou par cloner le depos avec la commande git

  ```bash
  git clone git@github.com:ranto-dev/LexiCorrect.git
  ```

- Puis, suivez les instructions de l'installation dans [step](./docs/steps.md)
